# Cover Prompts

## Prompt 1 - WeChat Cover

Create a clean, high-end tech editorial cover image for a WeChat Official Account article.

Main message: non-frontend roles can launch a professional React project with AI agents.

Visual style:

- dark editorial tech background
- credible engineering aesthetic, not flashy sci-fi
- layered code / project structure composition
- one primary panel labeled `apps/web`
- secondary floating panels labeled `ui`, `api-client`, `hooks`, `theme`
- subtle monorepo / package relationship cues
- premium, minimal, sharp typography
- no cartoon robots, no glowing brains, no generic AI icons

Text on image:

- Main title: `不会前端，也能起专业 React 项目`
- Subtitle: ``react-monorepo-init` for demo / MVP / internal tools`

Layout:

- 16:9
- bold Chinese title centered or upper-left
- supporting project panels around the title
- strong contrast, suitable for mobile thumbnail

## Prompt 2 - Inline Project Structure Figure

Create an inline editorial visual for a technical article showing the generated project structure of a professional React monorepo.

Visual style:

- code-editor inspired
- dark theme
- sharp typography
- organized tree structure
- visually highlight `apps/web` and `packages/*`
- make it feel professional and trustworthy
- no decorative gimmicks

Must include these labels:

- `apps/web`
- `packages/ui`
- `packages/api-client`
- `packages/shared`
- `packages/hooks`
- `packages/theme`
- `packages/config-eslint`
- `packages/config-typescript`
- `packages/config-tailwind`
- `packages/config-vitest`
- `.changeset`
- `.husky`
- `AGENTS.md`
- `DESIGN.md`
- `README.md`
- `CHANGELOG.md`

Caption intent:

`跑完之后，你拿到的不是一个页面，而是一套像样的前端项目。`
